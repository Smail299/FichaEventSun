
# Подробное описание:​
Этот плагин добавляет на сервер Ядро Солнца (Якорь возрождения) — специальный блок в центре PvP-арены, из которого через определённые промежутки времени выпадают ресурсы. Помимо этого, в радиусе 12 блоков от Ядра игроки получают монеты или опыт — тип награды меняется в среднем каждые 25 минут.

**Имеется:**

 • Два вида настройки лута, через плагин LootManager, или через сам плагин
 • Все полностью настраивается через конфиг
 • Имеется система, где если в радиусе Ядра находится несколько игроков, валюта делится поровну между ними.
